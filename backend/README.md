# GitTok Backend

FastAPI service for repository discovery, personalized feeds, slideshow generation,
contributors, and structured messaging requests.

## Run

```bash
cd backend
python3 -m uvicorn server.main:app --reload
```

## Environment

```env
DATABASE_URL=postgresql://...        # Required — Neon PostgreSQL
GITHUB_TOKEN=github_pat_...
FRONTEND_ORIGIN=http://localhost:3000
BACKEND_PROXY_SECRET=replace-me
TTS_PROVIDER=auto
OPENAI_API_KEY=sk-...
OPENAI_TTS_MODEL=gpt-4o-mini-tts
OPENAI_TTS_VOICE=alloy
CLOUDFLARE_API_TOKEN=your_cloudflare_api_token
CLOUDFLARE_ACCOUNT_ID=your_cloudflare_account_id
CLOUDFLARE_TTS_MODEL=@cf/deepgram/aura-2-en
CLOUDFLARE_TTS_SPEAKER=luna
```

`DATABASE_URL` must point at a PostgreSQL database (same Neon instance used by Prisma
for Clerk user sync). The backend creates its tables (`repo_snapshots`, `message_requests`)
on startup via `Base.metadata.create_all`.

Slide narration is generated by the backend and cached under `backend/cache/tts`.
`TTS_PROVIDER=auto` prefers Cloudflare Workers AI when `CLOUDFLARE_API_TOKEN`
and `CLOUDFLARE_ACCOUNT_ID` are present, otherwise it falls back to OpenAI when
`OPENAI_API_KEY` is present. Without either provider configured, the frontend
falls back to the browser speech engine.
